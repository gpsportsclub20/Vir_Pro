Question No. 3¶

In a football free kick, the height of the ball from the ground, \(h \ (x)\) m, is given by the equation below, where \(x\) is the horizontal distance of the ball from the player who takes the free kick.

Find the possible horizontal distance, in m, when the height of the ball is 8 m from the ground.

\[\begin{split}\begin{align*} h\ (x) &= -x^{2} + 6x\\ \end{align*}\end{split}\]

3.1 Sample Solution¶

When the height of the ball is 8 m from the ground,

\[\begin{split}\begin{align*} h\ (x) &= 8 \\ -x^{2} + 6x &= 8 \\ x^{2} - 6x + 8 &= 0 \\ (x - 2)(x - 4) &= 0 \\ x &= 2 \ or \ 4\\ \end{align*}\end{split}\]

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0,6,50)
y = -x**2 + 6*x

# Plotting the quadratic function
fig, ax = plt.subplots()
ax.plot(x,y,'r')
ax.set( xlabel='x',
        ylabel='y')

plt.xticks(np.arange(min(x),
                     max(x),
                     0.5))

plt.ylim([0,10])
plt.grid(axis='both',
         linestyle = '--')
plt.rcParams['figure.figsize'] = [15, 5]
plt.annotate("x = 2",[1.7,8.1])
plt.annotate("x = 4",[4.1,8.1])
plt.show()
